# Quick Start

Get up and running with Plugin Name in just a few minutes.

## Step 1: Open Settings

1. Open **Settings** (gear icon in the bottom left)
2. Navigate to **Plugin Name** in the left sidebar

You'll see configuration options for your plugin.

## Step 2: Configure Basic Settings

Set up your plugin preferences:

1. Go to the main settings tab
2. Configure the basic options according to your needs
3. Adjust any advanced settings if desired

## Step 3: Start Using the Plugin

Once configured, you can start using the plugin features:

1. Open the command palette (Ctrl/Cmd + P)
2. Search for plugin commands
3. Try out the main features
